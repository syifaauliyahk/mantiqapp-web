# MantiqApp

MantiqApp adalah aplikasi pembelajaran Ilmu Mantiq berbasis web yang dirancang untuk membantu pengguna memahami konsep dasar logika secara sederhana, interaktif, dan mudah digunakan. Aplikasi ini dikembangkan sebagai bagian dari tugas Mata Kuliah Kapita Selekta.

## Fitur Utama
- 📚 Materi Ilmu Mantiq (tasawwur, tashdiq, proposisi, qiyas)
- 🧠 Kalkulator Silogisme (menarik kesimpulan otomatis)
- 🔍 Validator Proposisi (menentukan jenis A, E, I, O)
- 📝 Kuis untuk menguji pemahaman

## Teknologi yang Digunakan
- HTML
- CSS
- JavaScript

## Cara Menjalankan
1. Download atau clone repository ini
2. Buka file `index.html` di browser
3. Aplikasi siap digunakan

## Tujuan
Aplikasi ini bertujuan untuk membantu pengguna dalam memahami konsep dasar Ilmu Mantiq serta melatih kemampuan berpikir logis melalui fitur yang disediakan.
